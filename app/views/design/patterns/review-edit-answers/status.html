HM Land Registry pattern

Inline edit answers

This experimental pattern has been designed to create a seamless user experience when changing inputted answers.

The timeline below shows the extent to which this experimental pattern has been tested, validated and approved.

Pattern included in release

Developers

23/09/2021

During the rebuild of the Business e-Services sign up process from Oracle in to Microsoft Dynamics, the development team included this pattern on the "Check your answers" pages.

Component review

Developers

22/05/2021

This pattern was presented to the development team and given the green light to include in the forthcoming build.

Usability testing

User research

19/05/2021

This update tested extremely well with users and was their preferred choice, compared to the existing solution.

Pattern creation

Design

17/05/2021

When the user clicks the "Change" button, a formfield appears on the same screen, allowing them to make changes to their data, without having to return to an early part of the sign up process.

User need identified

User research

17/05/2021

The sign up process to Business e-Services is extremely long for users. Using the standard GOV.UK pattern check answers would result in the user returning to a very early part of the journey, which often lead to frustration or confusion.
