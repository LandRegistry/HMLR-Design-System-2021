What is this pattern?

This pattern is an adapted version of the check answers GOV.UK pattern designed to allow users to directly make changes to the summary information.

Example

Examples

  • Animated

Why was this pattern needed?

Replicating the GOV.UK pattern, a change link appears at the end of each row to allow users to alter their inputs if the information displayed in the summary table is incorrect. If the link labelled change is clicked, the user will not be brought back to the original input screen to make a change, as is the case with the GOV.UK pattern. Instead, the answer will be replaced with a text input auto-populated with the original answer. This will enable the user to make the change to the input without being redirected from the summary page to create a more seamless user experience.

When to use

Using this pattern is appropriate in instances where the user needs to alter their original text inputs or to modify stored data. No testing has been carried out around the use of this pattern for changing multiple choice inputs such radios or checkboxes.
