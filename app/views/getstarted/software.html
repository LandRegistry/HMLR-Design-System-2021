Software

Information about common software used by designers and developers at HM Land Registry.

Mural

Mural is a website that enables teams to collaboratively work together in a shared and dynamic virtual environment. Each Mural board allows you add sticky notes, shapes, connectors, images and frameworks, which make it a useful tool for team exercises.

Trello

Trello is a project management application that can be used online or via the Trello app. By creating a Trello board, tasks can be added and assigned to the relevant team members. This enables teams to stay organised and monitor overall progress.

Slack

Slack is a communication platform that allows users to direct message and call colleagues. Slack also enables connection and collaboration with the wider HM Land Registry community through the use of channels that are each dedicated to a particular theme. For example, there is a designated channel for 'accessibility' where people might ask relevant questions or post useful content to share with others in this channel.

PluralSight

Pluralsight is a website offering a wide variety of online courses and tutorials.

Marvel

Marvel is the platform used to make tech support requests to the HM Land Registry I.T department.
