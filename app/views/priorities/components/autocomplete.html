Experimental component

Accessible autocomplete

What is this component?

This component has been designed to allow users to easily search extensive dropdown menus without having to scroll.

Example

Examples

  • Animated

Why was this component needed?

The Portal Refresh team working on View My Applications are introducing a new update that allows users to see applications lodged by anyone in their organisation. This means that the user potentially has access to hundreds of different user applications.

To allow the user to view all of the applications for a particular individual, a dropdown menu was introduced. This could mean potentially scrolling through a list of hundreds of colleagues. By implementing search functionality within the dropdown menu itself, the user is able to locate a colleague quickly by typing the start of their name. As the user types, the search starts filtering the names in the dropdown menu.

The DRS team are using this component to allow users to select a transaction from a searchable dropdown menu.
