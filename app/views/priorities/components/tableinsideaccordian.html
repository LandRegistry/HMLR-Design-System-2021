Table inside accordian

What is this component?

The table inside accordion combines the functionality of the accordion component, allowing users to hide and show sections of related content, with the functionality of the table component, allowing users to scan and compare information.

Why was this component needed?

This component helps avoid a situation where the user is inundated with content when they first land on this page.

By grouping relevant content in an accordion, summary information can be added to each accordion label to allow a basic overview to be provided at first glance. This will allow the user to easily identify and reveal the content relevant to them without the need to look through extensive additional information. If a particular section is of interest, the user can click the accordion to expand the content and receive the full breakdown of information.

Interactive table inside accordian

The table can also be adapted to include suitable interactive elements such checkboxes.
