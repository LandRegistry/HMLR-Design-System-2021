Using GDS

Why can the GDS Transport font not be used across all HMLR services?

As per the advice provided through the GOV.UK Design System, the GDS Transport Font can only be used on service.gov.uk subdomain. Outside of the service.gov.uk subdomain, an alternative font must be used. The default font for HMLR is Roboto.

Can I use GDS components if the HMLR service is not being hosted on the service.gov.uk subdomain?

Yes, components taken from the GOV.UK Design System can and should be used on all HMLR services.

Are adaptations allowed to be made to approved GDS components?

Adaptations to GDS components should be informed and driven by user research. If a particular user need has resulted in a team adapting an existing GDS component, please contact the Design System Team as it may be of interest to others in the community.

How can I edit a GDS component using the prototype kit?

Custom changes can be made by adapting the HTML and CSS of the original components.

Should HMLR services adhere to the same standards of accessibility as those outlined by GDS?

Yes, HMLR must adhere to the government's standards around accessibility.
